# 🤝 Contributing

Thanks for your interest in improving **Agent Template**.

We welcome:
- documentation improvements
- better examples
- stronger safety defaults
- better prompt structures
- additional tool templates
- additional API binding templates
- improved tests and developer experience

---

## ✅ Contribution guidelines

### Keep changes structured
Please keep the repository modular and easy to understand.

### Prefer explicit files over hidden logic
Do not hide core rules only in code comments or chat instructions.

### Keep safety first
If a contribution affects:
- tool execution
- approvals
- external write actions
- auth
- secrets
- PII handling

please document it clearly.

### Improve docs when changing behavior
If you change how the template works, update the relevant docs too.

---

## 🧪 Before opening a PR

Please make sure:
- the structure still makes sense
- filenames are consistent
- docs are updated
- the change improves reuse
- risky actions are still approval-aware

---

## 📬 Suggested contribution types

- README improvements
- new prompt packs
- better test templates
- new API integration scaffolds
- monetization guides
- deployment examples
- best-practice updates

---

## 💡 Philosophy

This repository exists to make AI agent creation:
- repeatable
- cleaner
- safer
- easier to hand off
- easier to scale

If your contribution supports that goal, it is likely a good fit.
